## 简介

就目前来看，大部分 ChatGPT 相关的工具本质上就是调整 prompt， AI Anything 可以在只编写结构化数据的情况下生成工具，同时还支持使用函数来生成更复杂的 prompt

## 💻 发展

- 克隆此代码库
- 使用 `corepack enable` 启用 Corepack (对于 Node.js < 16.10，请使用`npm i -g corepack`)
- 使用 `pnpm install` 安装依赖项
- 使用 `pnpm dev` 运行交互测试
